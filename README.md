# keiba-predictor v3.1

GUIベースの競馬予想ソフトです。デフォルトは実データ取得モードです。

## 起動手順（Windows）
1. `setup.bat` を初回のみ実行
2. `start.bat` で起動

## 重要
- デフォルトは実データ取得（JRA/NAR）を試行します。
- 接続に失敗した場合は「実データの取得に失敗しました。ネットワーク接続を確認してください」と表示して停止します。
- MockSourceは `pytest tests/` と、設定画面の「オフラインモード（テスト用）」有効時のみ使用してください。
- 起動に失敗した場合、`start_error.log` にエラー内容を出力します。

## DataLab将来拡張
DBスキーマには将来差し替え用の項目（調教・血統・時系列オッズ等）を保持可能な設計を含みます。
