# keiba-predictor v3.1

GUIベースの競馬予想ソフトです。実データ取得モードのみをサポートします。

## 起動手順（Windows）
1. `setup.bat` を初回のみ実行
2. `start.bat` で起動

## 重要
- 起動時に JRA/NAR の実データ取得を試行します。
- 取得に失敗した場合は「実データの取得に失敗しました。ネットワーク接続を確認してください」と表示して処理を停止します。
- 起動に失敗した場合、`start_error.log` にエラー内容を出力します。
- 起動時は先にメインウィンドウを表示し、その後に当日データ取得を開始します。
